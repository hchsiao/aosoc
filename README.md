## Usage

Download dependency: 
```
./download.sh
```

Simulation without debugger (DO NOT USE): 
```
cd run
make inner-ncsim V=1 JTAG_ENABLE=0
```

Simulation with debugger + gdb: 
```
cd run
make run V=1
```

Simulation with debugger + gdb: 
```
cd run
make sim V=1
```
